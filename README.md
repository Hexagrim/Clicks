# Clicks
A 2d clicking game made in a short time to test out Unity UI elements.

## Button
The button is simply labled Click. Its the only thing you see when starting the game alongside the shop icon.
On press it gives 1 point and theres a chance of hitting a multiplier which is 2x in the base form.
Theres also a slight delay when spamming the button.

All of the aspects are upgradable inculding the delay in the shop for set clicks.

## Shop
The shop has mainly 4 items currently.

1. More clicks per press
2. More multiplier
3. More chance of getting a multiplier
4. Less delay when clicking

## Thats about it. Just click and try to get the highest score!
